# ecommerce-html-mock

Mock de e-commerce em HTML/CSS/JavaScript construído como projeto da disciplina de front-end. Sem backend — toda a navegação e o "carrinho" rodam no cliente.

## O que demonstra
- Estrutura semântica em HTML.
- Layout responsivo em CSS.
- Manipulação de DOM em JavaScript vanilla (carrinho, busca, filtros).

## Como abrir
Basta abrir `index.html` no navegador.

## Status
Projeto acadêmico — referência de marcação e estilo.
